# ClimateScope-Dashboard-R-Shiny-Analyses-multivari-es
Ce projet vise à analyser les principaux enjeux liés au changement climatique à travers un ensemble d’indicateurs climatiques, socio-économiques et démographiques. Il a été réalisé dans le cadre de la SAÉ Modélisation statistique pour les données complexes et le Big Data du BUT Science des Données
